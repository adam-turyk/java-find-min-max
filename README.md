# Find Min/Max Program (Java Console Application)

This is a Java console-based program that finds the highest and lowest numbers from a set of user-input values.

## Features
- Accepts multiple numbers from the user using an array
- Finds the maximum (highest) number
- Finds the minimum (lowest) number
- Uses a menu system to allow the user to choose between finding max or min
- Continues running until the user chooses to exit

## Concepts Used
- Arrays
- Loops (for and while loops)
- Methods
- Conditional statements (if statements)
- User input with Scanner
- Basic algorithm logic (comparison of values)

## Purpose
This project was created to practice core Java programming fundamentals, especially working with arrays, loops, and simple algorithms for processing data.

## Future Improvements
- Allow user to enter any number of values (not fixed array size)
- Add sorting functionality (ascending/descending)
- Improve input validation for non-numeric entries
